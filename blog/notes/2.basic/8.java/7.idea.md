## The Tomcat connector configured to listen on port 8080 failed to start

```js
1. 调出命令终端
win+r 并且输入cmd
 
2. 查看某个端口(8080)的被占用情况
netstat -ano |findstr 8080
 
3. 根据进程号(24068)关闭进程即可
taskkill -PID 24068 -F 
```
