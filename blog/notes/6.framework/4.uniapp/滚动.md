## 滚动底部触发事件

滚动到底部/右边，会触发 scrolltolower 事件

```html
 <scroll-view class="main_box" scroll-y="true" @scrolltolower="lower">
 </scroll-view>
```

## 滚动顶部

`onPageScroll`监听页面滚动

`uni.pageScrollTo`设置滚动到的位置和动画时长

```js
<script>
  export default {
    data() {
      return {
        showBackTop: false
    },
    onPageScroll(e) {
      this.showBackTop = e.scrollTop > 500 ? true : false;
    },
    methods: {
      scrollTop() {
        uni.pageScrollTo({
          scrollTop: 0,
          duration: 100
        })
      }
    }
  }
</script>
```
