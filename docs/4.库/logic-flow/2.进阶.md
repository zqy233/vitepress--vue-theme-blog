## 点击事件

### 节点点击

```js
 this.lf.on('node:click', ({ data }) => {
   this.currentNode = data
  })
```

### 点击画布空白处

```js
  this.lf.on('blank:click', ({ data }) => {
   this.currentNode = null
  })
```
